### Buat sebuah Aplikasi Baru {#buat-sebuah-aplikasi-baru}

1. `blitz new myAppName`
2. `cd myAppName`
3. `blitz dev`
4. Lihat aplikasi baru kamu di
   [http://localhost:3000](http://localhost:3000)

Komunitas Blitz itu ramah, aman, beragam, inklusif, dan menyenangkan! LGBTQ+,
wanita, dan minoritas sangat diterima.

[Bergabung dengan komunitas Discord kami](https://discord.blitzjs.com) dimana kita saling membantu
untuk membangun aplikasi Blitz. Disini juga kami saling berkolaborasi membangun
Blitz itu sendiri.

Untuk pertanyaan dan berdiskusi panjang,
[buatlah sebuah artikel diskusi di forum kami.](https://github.com/blitz-js/blitz/discussions)

Untuk pengantar selengkapnya, bacalah
[Bagaimana Komunitas Beroperasi](./how-the-community-operates). pengantar ini merinci segalanya, 
termasuk bagaimana caranya meminta bantuan, bagaimana melaporkan sebuah bugs, bagaimana menyarankan sebuah
fitur baru, dll.

## Langkah Selanjutnya {#langkah-selanjutnya}

### Tutorial {#tutorial}

[Tutorial](./tutorial) Tutorial adalah panduan lengkap melalui semua bagian fundamental Blitz, 
termasuk menambahkan model ke database kamu dan membaca serta memperbarui data dari frontend-mu.

### Mempelajari {#mempelajari}

Dan inilah aspek utama Blitz yang ingin kamu ketahui.

- Bagaimana caranya [membuat sebuah Halaman](./pages)
- Bagaimana caranya [menggunakan Routing File-System](./routing)
- Bagaimana caranya menyiapkan dan [menggunakan sebuah Database](./database-overview)
- Bagaimana caranya menggunakan [Queries pada Blitz](./query-resolvers) dan
  [Mutations](./mutation-resolvers) untuk membaca dan menulis sebuah data ke dalam database-mu
- Bagaimana cara men-scaffold semua kode untuk model database dengan perintah
  [`blitz generate`](./cli-generate)
